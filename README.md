## 教学文件的运行方式

1、安装所有必要的python库，并在GPU环境下运行。

2、解压 img_set.zip 压缩包

3、运行 Jupyter note book 指导文件时，务必依次执行程序。根据自己电脑算力，调整epochs的数值。

4、遇到报错时，查看错误提示，一般是未定义，这时需要重新导入库。若需要单独运行某个模块时，其依赖的函数和类也要提前运行。
